## Webhook Configurations

Webhooks enable real-time data sharing. Configure them to trigger events like page updates or new comments.

### Setup Steps

<Steps>
  <Step title="Create Webhook" icon="plus">
    Navigate to Settings > Integrations > Webhooks. Click "New Webhook" and enter your endpoint URL, such as `https://your-webhook-url.com/webhook`.
  </Step>

  <Step title="Configure Events" icon="settings">
    Select events like `page.updated` or `comment.created`. Test the configuration with the built-in simulator.
  </Step>

  <Step title="Secure Endpoint" icon="shield">
    Add HMAC signature verification using your secret key.
  </Step>
</Steps>

### Sample Payloads

Handle incoming payloads in your server.

<CodeGroup tabs="Node.js,Python">
  ```javascript
  const express = require('express');
  const app = express();
  app.use(express.json());

  app.post('/webhook', (req, res) => {
    const event = req.headers['x-event-type'];
    const signature = req.headers['x-signature'];
    console.log('Event:', event, 'Payload:', req.body);
    res.status(200).send('OK');
  });

  app.listen(3000);
  ```
  ```python
  from flask import Flask, request, jsonify

  app = Flask(__name__)

  @app.route('/webhook', methods=['POST'])
  def webhook():
      event = request.headers.get('X-Event-Type')
      signature = request.headers.get('X-Signature')
      print(f"Event: {event}, Payload: {request.json}")
      return jsonify({'status': 'OK'}), 200

  if __name__ == '__main__':
      app.run(port=3000)
  ```
</CodeGroup>

<ParamField path="event" param-type="string" required="true">
  The type of event triggered, e.g., `page.updated`.
</ParamField>

<ParamField header="X-Signature" param-type="string" required="true">
  HMAC SHA256 signature for verification.
</ParamField>

## Embedding External Resources

## Export and Import Options

Maintain backups or migrate content seamlessly.

<Expandable title="Advanced Export Formats" default-open="false">
  Export to Markdown, PDF, or HTML. Use the API for bulk operations:

  <Request tabs="cURL,JavaScript">
    ```bash
    curl -X POST https://api.example.com/v1/export \
      -H "Authorization: Bearer YOUR_API_KEY" \
      -H "Content-Type: application/json" \
      -d '{"format": "markdown", "pages": ["all"]}'
    ```

    ```javascript
    const response = await fetch('https://api.example.com/v1/export', {
      method: 'POST',
      headers: {
        'Authorization': 'Bearer YOUR_API_KEY',
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({
        format: 'markdown',
        pages: ['all']
      })
    });
    const data = await response.json();
    ```
  </Request>
</Expandable>

<Callout kind="alert">
  Always use HTTPS endpoints and verify signatures to protect your data.
</Callout>
